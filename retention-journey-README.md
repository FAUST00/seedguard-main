Preview and usage

Files added:
- retention-journey.html — standalone drop-in section using Tailwind CDN

Quick preview

1. Open `retention-journey.html` directly in a browser.

2. (Recommended) Serve from a local HTTP server and open http://localhost:8000/retention-journey.html

Python 3 (cross-platform):

```bash
python -m http.server 8000
```

Windows (if `python` maps to another version):

```bash
py -m http.server 8000
```

Notes
- The file uses the Tailwind CDN; no build step required for preview.
- To integrate into your site, copy the section markup into the desired page or component (ensure Tailwind is enabled in your project).
